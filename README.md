# Tlash
## A Clash(Clash.Meta) client wriiten in rust, egui

-----

### Build:
1. Install build dependencies:
    1. git, rust(rustup)
2. `git clone https://github.com/hellopoisonx/tlash.git`
3. `cd tlash && cargo build --release && cargo install .`
